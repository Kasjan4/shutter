# Shutter

## Overview

A web application made in React Hooks which consumes the Pixabay API to showcase photos and videos.

[CHECK IT OUT HERE ](https://kasjanhinc.github.io/shutter/)

**Technologies used:**

- HTML5
- CSS3
- JavaScript ES6
- React Hooks, Router
- React Reveal
- Pixabay open source API
- Git
- GitHub
- Font Awesome
- Pixlr editor
- Google Fonts
- Royalty free graphics
